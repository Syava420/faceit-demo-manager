# 📥 FACEIT Demo Hub

**FACEIT Demo Hub** (бывший *faceit-demo-manager*) — это современное десктопное приложение для Windows (WPF / C#), разработанное для автоматизации управления записями матчей (демками) CS2 и CS:GO, скачанными с платформы FACEIT.

Приложение избавляет от необходимости вручную распаковывать архивы, переименовывать файлы и искать статистику сыгранных матчей.

---

## ✨ Ключевые возможности

* 🔄 **Автоматический импорт и распаковка**: Отслеживает папку `Downloads` (Загрузки) в реальном времени, автоматически распаковывает сжатые архивы `.dem.zst` с помощью встроенной утилиты `zstd` и переименовывает файлы по удобному шаблону.
* 📊 **Интеграция с API FACEIT**: Автоматически запрашивает информацию о матчах по никнейму игрока — счет, карту, дату, индивидуальный K/D/A и ADR.
* 📂 **Библиотека демок**: Удобная таблица с поиском, сортировкой, фильтрацией и возможностью распределять записи по пользовательским категориям (папкам) и добавлять заметки к играм.
* 🎮 **Быстрый запуск CS2**: Запуск игры напрямую из приложения в безопасном режиме (`-insecure`) сразу на нужную демку с поддержкой автоматического воспроизведения.
* ⌨️ **Управление биндами**: Настройка горячих клавиш для комфортного просмотра демок (управление скоростью, пауза, переключение игроков, подсветка X-Ray).
* 🎨 **Современный UI**: Полностью темная тема с поддержкой DPI-scaling (четкие шрифты на любых мониторах) и плавными скроллбарами.

## 🚀 Установка и запуск

Самый простой способ начать использовать программу — скачать готовый установщик:

1. Перейдите в раздел **[Releases](https://github.com/Syava420/faceit-demo-manager/releases)**.
2. Скачайте файл **`Setup.exe`**.
3. Запустите установщик — он автоматически распакует программу и утилиту `zstd.exe` в папку `AppData`, а также создаст ярлык на рабочем столе.

## 🛠️ Сборка из исходников

Проект спроектирован так, чтобы компилироваться без использования тяжелых сред разработки вроде Visual Studio или MSBuild. Разметка интерфейса хранится в виде XAML-строки и парсится динамически при запуске.

Для компиляции вам понадобится только стандартный компилятор C# (`csc.exe`), который по умолчанию встроен во все версии Windows с .NET Framework 4.0+.

### Инструкция:
1. Откройте PowerShell в папке проекта.
2. Выполните скрипт компиляции:
   ```powershell
   powershell -ExecutionPolicy Bypass -File .\compile.ps1
   ```
3. После успешного завершения исполняемый файл **`FaceitDemoManager.exe`** появится в папке проекта.

> ⚠️ **Примечание**: Для корректного импорта демок рядом с `FaceitDemoManager.exe` должен находиться файл `zstd.exe` (входит в исходные файлы репозитория).
